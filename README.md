# easyCircos
A web interface to build Circos images

Requirements
============
Node, mongo (with a database named easyCircosDB and a user easyCircos)

Install
=======

```
git clone git@github.com:nicgirault/easyCircos.git
git submodule init
git submodule update
bower install
npm install
```

Run
===

```
gulp server
```
